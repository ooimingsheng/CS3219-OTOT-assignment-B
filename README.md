# CS3219-OTOT-assignment-B

### Basic setup
1. Setup postgres database
2. Install yarn globally on your computer
3. git clone project
4. Go to project directory
5. Run `yarn setup`


### Test backend api
1. cd server
2. yarn test


### Start the backend from terminal
1. cd server
2. yarn start


### Start the frontend from terminal
1. cd client
2. yarn start
